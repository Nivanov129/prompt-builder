# Prompt Builder — план

## Что делаем
Веб-приложение (SPA) для создания промптов — аналог «МАЯК ОКО», но лучше.
Хостинг: GitHub Pages. Стек: чистый HTML/CSS/JS (zero dependencies).

## Чем лучше МАЯК ОКО
1. **Превью в реальном времени** — промпт собирается на лету при вводе
2. **Умные шаблоны** — разные структуры для разных типов задач (текст, код, анализ, изображение)
3. **Few-shot примеры** — кнопка «добавить пример» для вставки пар input/output
4. **Chain-of-thought** — опциональный тогл «пошаговое мышление»
5. **Пресеты** — готовые заполненные шаблоны для быстрого старта
6. **Копирование в 1 клик** + кнопки для открытия в ChatGPT/Claude/etc
7. **Мобильная версия** работает
8. **Тёмная тема**

## Структура промпта (поля)
- **Роль** — кто ты (эксперт, специалист)
- **Задача** — что нужно сделать (главное поле)
- **Контекст** — вводные данные, ситуация
- **Аудитория** — для кого результат
- **Формат** — как оформить (длина, стиль, структура)
- **Ограничения** — чего избегать, лимиты
- **Примеры** (опционально) — пары вход/выход для few-shot
- **Chain of Thought** — тогл: «сначала проанализируй, потом ответь»

## Шаблоны задач
- 📝 Текст (пост, статья, письмо)
- 💻 Код (написать, отрефакторить, ревью)
- 📊 Анализ (данные, сравнение, исследование)
- 🎨 Изображение (промпт для DALL-E/Midjourney/Flux)
- 🧠 Брейншторм (идеи, планирование)
- 📋 Разное (универсальный)

## UI
- Левая панель: поля ввода (scrollable)
- Правая панель: live preview промпта (sticky)
- Сверху: выбор шаблона (табы/карточки)
- Мобилка: поля сверху, превью снизу (toggle)
- Кнопки: Скопировать, Сбросить, открыть в ChatGPT/Claude/Gemini/DeepSeek
- Тёмная/светлая тема (toggle)

## Дизайн
- Минималистичный, современный
- Glassmorphism accents
- Цветовая схема: тёмная по умолчанию, нейтральные тона + accent color
- Шрифт: system font stack
- Анимации: subtle transitions
